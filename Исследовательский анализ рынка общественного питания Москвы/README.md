#### Описание проекта
Исследование направлено на поиск оптимального варианта для открытия нового заведения в г.Москва
#### Навыки и инструменты
---
- python
- pandas
- matplotlib
- seaborn
- phik
---
#### Общий вывод
В ходе исследования проанализировали данные 8406 заведений общественного питания Москвы за лето 2022 года. Данные содержат: название, адрес, расположение, рейтинг, принадлежность к сети, количесвто посадочных мест, часы работы и информацию о среднем чеке в заведении. На основе данных сделаны выводы о текущем состоянии рынка общ. питания Москвы и даны рекомендации по открытию нового заведения в разрезе типа и категории места, а также его расположения
