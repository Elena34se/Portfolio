#### Описание проекта
Исследовательский анализ изменения пользовательских предпочтений и популярности событий сервиса Яндекс-Афиша осенью 2024 года
#### Навыки и инструменты
---
- Исследовательский анализ данных
- Статистический анализ данных
- python
- pandas
- matplotlib
- seaborn
- A/B-тесты
- statsmodels.stats
- ttest_ind (t-test с поправкой Уэлча)
- mannwhitneyu (тест Манна-Уитни)
- Jupyter Notebook
---
#### Общий вывод
- В ходе исседования проведен **анализ 21422 пользователей**, совершающих заказы на сервисе Яндекс-Афиша в период **с 1 июня по 31 октября 2024 года**. В ходе исследования изучены сезонные изменения в заказах, осенняя активность пользователей и популярные события и сервисы-партнеры по продаже билетов. 
- В ходе **статистического анализа** были **проверены гипотез о разнице между пользователями мобильных и стационарных устройств**. Проведены A/B - тесты с участием 2 групп: группа мобильных и группа стационарных устройств. В качестве целевой метрики для первой гипотезы выбрано **среднее количество заказов на пользователя**. Для второй гипотезы - **среднее время между заказами**.
- По результатам исследования даны **рекомендации по улучшению работы сервиса**
