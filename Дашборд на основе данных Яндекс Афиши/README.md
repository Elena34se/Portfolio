## Дашборд на основе данных Яндекс Афиши
---
[Яндекс-афиша-datalens](https://datalens.yandex/xewbnjxzwobmh)
#### Описание проекта
Дашборд содержит данные сервиса Яндекс-Афиша за период с июня по октябрь 2024г. **Цель дашборда** - визуализация динамики ключевых бизнес-показателей для анализа популярности мероприятий и структуры выручки в разрезе категорий событий и устройств.
#### Навыки и инструменты
---
- Yandex DataLens
- Создание датасета
- Wizard-чарты
- Расчет бизнес-показателей
- Настройка селекторов
- Разработка и настройка дашборда
