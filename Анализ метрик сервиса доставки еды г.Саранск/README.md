## Анализ метрик сервиса доставки еды г.Саранск (Дашборд)
---
[доставка-еды-datalens](https://datalens.yandex/hq6sku45axza1)
#### Описание проекта
Дашборд содержит информацию о пользовательской активности сервиса по доставке еды г.Саранск за периолд с мая по июнь 2021г. **Цель дашборда** - визуально отразить состояние клиентской базы в городе.
#### Навыки и инструменты
---
- Yandex DataLens
- SQL-запросы
- QL-чарты
- Расчет бизнес-метрик (LTV, DAU, CR, RR, AOV)
- Разработка и настройка дашборда
